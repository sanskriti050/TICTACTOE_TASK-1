# 🎮 Tic Tac Toe Game

A classic Tic Tac Toe game developed using **Java Swing**. This project provides a simple and interactive graphical interface where two players can play the game on a 3×3 board.

---

## 📌 Features

- 🎨 Java Swing GUI
- 👥 Two Player Mode
- ❌⭕ X and O Turn Management
- 🏆 Winner Detection
- 🤝 Draw Detection
- 📊 Live Scoreboard
- 🔄 Restart Game
- 🆕 New Game
- 🚪 Exit Button
- 🌙 Dark Theme UI
- 🟢 Winning Cells Highlight

---

## 🛠 Technologies Used

- Java
- Java Swing
- AWT
- OOP (Object-Oriented Programming)

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/TicTacToe.git
```

2. Open the project in IntelliJ IDEA, Eclipse, or VS Code.

3. Compile the Java files.

```
javac *.java
```

4. Run the project.

```
java Main
```

---

## 🎮 Gameplay

- Player X starts first.
- Players take turns placing X and O.
- First player to align three symbols horizontally, vertically, or diagonally wins.
- If all cells are filled without a winner, the game ends in a draw.
